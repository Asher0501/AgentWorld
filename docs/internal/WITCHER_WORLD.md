# 猎魔人世界：「白果园」

> AgentWorld 的默认域（domain）——基于《猎魔人》系列改编的世界设定。
> 所有数据由 `config/domain.json` 驱动，引擎代码与背景无关。

---

## 世界概览

白果园（White Orchard）及其周边——泰莫利亚王国在战后复苏中的广袤舞台。战争虽已结束，但余波仍在：魔力流动紊乱，怪物出没，生计艰难。

**时间起点**: 春·第 1 天 08:30

**时间推进**:
- 白天（06:00–21:00）：每 tick **30 分钟**
- 夜间（21:00–06:00）：每 tick **6 小时**

## 属性系统

| 属性 | 范围 | 描述 |
|:----|:----|:------|
| `vitality` | 0–100 | 体力。行动消耗，休息恢复 |
| `satiety` | 0–100 | 饱腹度。随时间衰减，进食补充 |
| `mood` | 0–100 | 心境。社交/交易提升，独处/失败降低 |

### 自然衰减

每个 tick：
- 体力：-3（基础）+ 移动额外消耗
- 饱腹度：-5（基础）
- 心境：±0（无事件不变化）

### 属性警戒线

```
体力 < 30: 极度疲劳，必须休息
体力 < 15: 濒临昏倒，任何行动都极度困难
饱腹 < 30: 极度饥饿，必须进食
饱腹 < 15: 饥渴，无法集中精力
心境 < 20: 极度低落，可能拒绝社交
```

警戒值注入 LLM #1 prompt，作为 NPC 自主决策的参考。

## 配置源

所有域特定配置集中在 `config/` 目录下：

```
config/
├── domain.json           # 实体定义、prompt 模板、配方、校验 mask
├── domain.schema.json    # domain.json 的 JSON Schema
├── node_config.json      # 节点类型本体：前缀映射、规则
├── world/seeds/          # seed 数据（初始 NPC/区域/物品配置）
│   ├── characters.json
│   ├── items.json
│   └── zones.json
└── domain_update.py      # 生成 domain.json 的脚本

运行时读取优先级：
  1. config/domain.json 中的 seed_paths → 加载各 seed
  2. prompt_slots → 定义 LLM prompt 模板槽位
  3. verification_mask → 控制校验项启停
  4. node_ontology → 节点类型分类和约束
```
