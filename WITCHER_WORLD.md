## 世界概览

白果园（White Orchard）——泰莫利亚王国的一个边陲小镇。战争虽已结束，但余波仍在：魔力流动紊乱，怪物出没，生计艰难。

**时间起点**: 春·第 1 天 08:00（纪元 1272 年）

**时间推进**:
- 白天（06:00–21:00）：每 tick **30 分钟**
- 夜间（21:00–06:00）：每 tick **6 小时**


## 任务与行为模式

### 典型 NPC 决策逻辑

LLM #1 在 prompt 中看到：
1. 自己的属性状态（活力/饱食/心境）
2. 当前位置和同区域其他人
3. 库存物品
4. 性格标签
5. 基本生存需求警戒描述

然后自主决定做什么。例如：
- 杰洛特低饱食 → "先去酒馆吃点东西，顺便打听猎魔委托"
- 哈托里有矿石需求 → "看看卓尔坦的矿石质量，价格合适就买"
- 希里低饱食 → "不管维瑟米尔说什么，先去买面包吃"

## 配置驱动

所有猎魔人世界特定内容定义在 `src/agent_world/config/domain.json`：

```json
{
  "world_name": "Witcher",
  "zones": [...],
  "recipes": [...],
  "npc_initial_zones": {...},
  "adapter": {
    "survival_needs": "...",      // 生存需求描述（含 {v}/{s}/{m} 模板）
    "entity_identity": "...",     // 角色身份模板
    "personality_format": "...",  // 性格标签格式
    "system_role": {...},         // 各 LLM 阶系统提示词
    "output_format": {...},       // 各 LLM 输出格式指令
    ...
  }
}
```

**切换世界观只需替换 `domain.json` + 同步 `node_config.json`**，无需改动任何 Python 代码。
